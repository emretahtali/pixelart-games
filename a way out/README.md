this is a game i made in 2020.

it was made in 72 hours for ludum dare 46 game jam.

it's made in GameMaker Studio 2.

---
controls:

WASD / Arrow Keys - move and jump

E / X - interact

Shift - hold on to the walls

R - restart the room

G - restart the game

F - fullscreen on/off

Esc - go back
